2. Drag and drop the `radiantoptimizer` folder
3. Your site will be live in seconds!

### Custom Domain
To use your custom domain (radiantoptimizer.com):
1. Deploy to your hosting provider
2. Update DNS settings:
   - Add A record pointing to your host's IP
   - Or add CNAME record as directed by your host
3. Wait for DNS propagation (up to 48 hours)
